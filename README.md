# taotao-shopping

简介
=======
综合性的B2C网站，实现的功能主要有：前端页面展示，后台商品管理，商品搜索，会员登录，添加购物车，订单结算等。

* taotao-common
* taotao-manager
  * taotao-manager-mapper：逆向工程生成，适用于单表操作
  * taotao-manager-pojo
  * taotao-manager-service
  * taotao-manager-web
* taotao-order
* taotao-parent
* taotao-portal
* taotao-rest
* taotao-search
* taotao-sso

技术选型
==========
* Spring, SpringMVC, Mybatis
* JSP、JSTL、jQuery、jQuery plugin、EasyUI、KindEditor（富文本编辑器）、CSS+DIV
* Rdeis-缓存服务器
* Solr-搜索
* httpClient-调用系统服务
* MySQL
* Nginx-web服务器
